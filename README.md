# Laboratorio 19 - JavaScript: Consumo de APIs
**Autor:** Mark Hancco

## Descripción
Ejercicios de consumo de la API de PokéAPI usando JavaScript.

## Ejercicios Implementados

### Ejercicio 3: Obtener nombre de Pokémon por ID
- **Archivo:** `ejercicio3.html`
- Permite ingresar un ID y muestra el nombre del Pokémon

### Ejercicio 4: Altura y peso de Pikachu usando .then
- **Archivo:** `ejercicio4.html`
- Obtiene datos de Pikachu usando promesas con .then()

### Ejercicio 5: Altura y peso de Pikachu usando async/await
- **Archivo:** `ejercicio5.html`
- Obtiene datos de Pikachu usando async/await

### Ejercicio 6: URL del sprite de Charizard
- **Archivo:** `ejercicio6.html`
- Muestra la URL del sprite front_default de Charizard

### Ejercicio 7: Listar primeros 20 Pokémon
- **Archivo:** `ejercicio7.html`
- Lista los primeros 20 Pokémon desde la API

### Ejercicio 8: Pokémon aleatorio
- **Archivo:** `ejercicio8.html`
- Genera un número aleatorio del 1-898 y muestra ese Pokémon

### Ejercicio 9: Buscador visual de Pokémon
- **Archivo:** `ejercicio9.html`
- Buscador completo que muestra: imagen, nombre, ID, peso, altura y habilidades

## Instrucciones de uso
Abrir cada archivo HTML en el navegador para probar los ejercicios.

## API utilizada
